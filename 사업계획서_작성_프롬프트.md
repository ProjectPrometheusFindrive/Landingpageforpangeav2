# Pangea 사업계획서 작성 프롬프트
## 창업진흥원 예비창업자 패키지사업 지원용

---

## 📋 작성 가이드라인

**목적:** 렌터카 리스크 관리 플랫폼에서 시작하여 미래 로봇택시 운영 플랫폼으로 성장하는 비전을 설득력 있게 제시

**핵심 메시지:** 
- 현재: 렌터카 사업자의 디지털 전환 및 리스크 관리 자동화
- 미래: 자율주행 시대의 운수사업 운영 플랫폼

**작성 톤앤매너:** 
- 데이터 기반의 객관적이고 논리적인 서술
- 기술적 전문성과 시장 이해도를 균형있게 표현
- 단계적 성장 전략의 실현 가능성 강조

---

## 1. Vision (비전)

### 작성 프롬프트:

```
"미래 운수사업 운영의 자동화"라는 비전을 다음 구조로 작성하세요:

[현재 → 미래로의 여정]
1. **현재 (2026-2028)**: 렌터카 사업자의 디지털 전환
   - 수기 관리 → 자동화 관리 시스템 구축
   - 개별 사업자 → 플랫폼 생태계
   - 사후 대응 → 예측 기반 사전 관리

2. **중기 (2028-2030)**: 운수사업 전반으로 확장
   - 렌터카 → 택시, 화물, 버스 등 전체 운수업
   - 리스크 관리 → 종합 운영 플랫폼
   - 국내 시장 → 글로벌 진출

3. **장기 (2030+)**: 자율주행 시대의 인프라
   - 유인 차량 → 로봇택시 운영 플랫폼
   - 중소 사업자도 자율주행 서비스 운영 가능한 인프라 제공
   - 제조사/대기업 독점 방지, 민주적 모빌리티 생태계 구축

[핵심 가치 제안]
- "누구나 운수사업을 체계적으로 운영할 수 있는 세상"
- "자율주행 시대, 중소 사업자도 경쟁할 수 있는 공정한 플랫폼"
- "데이터와 자동화로 운영 효율성 극대화"

[예상 임팩트]
- 운수사업자 ○○만 명의 디지털 전환
- 로봇택시 시대 중소 사업자 생존 가능성 확보
- 모빌리티 산업의 민주화
```

**작성 팁:**
- 테슬라의 "지속가능한 에너지로의 전환" 같은 명확한 미션 스타일
- 단계별 마일스톤을 구체적으로 제시
- 왜 이 비전이 중요한지 사회적 의미 강조

## 2. 문제현황 (Problem)

### 작성 프롬프트:

```
렌터카 사업자가 직면한 문제를 다음 3단계 구조로 작성하세요:

[문제 1단계: 현재의 운영 비효율]
1. **차량 도난 대응**
   - 현황: 도난 발생 후 평균 ○○시간 후 인지
   - 문제: 실시간 위치 추적 불가, 수동 확인 의존
   - 손실: 연간 ○○억원 규모 피해 (통계 자료 인용)

2. **차량 구매 대출 확보의 어려움**
   - 현황: 우량 사업자도 담보/신용 부족으로 대출 거절률 ○○%
   - 문제: 실제 운영 데이터 기반 평가 시스템 부재
   - 결과: 사업 확장 기회 상실, 고금리 대출 의존

3. **사고 및 보험사기 관리**
   - 현황: 연간 렌터카 사고율 ○○%, 보험사기 ○○건
   - 문제: 사전 예방 불가, 사후 대응에만 집중
   - 손실: 보험료 증가, 차량 가동률 하락

4. **과태료 및 미납금 관리**
   - 현황: 사업자당 평균 ○○건의 미납/체납 발생
   - 문제: 수기 관리, 추적 어려움, 회수율 저조
   - 손실: 연간 ○○억원 미회수금 발생

[문제 2단계: 수기 관리의 한계]
- **현재 운영 방식**: 엑셀, 수기 장부, 개별 전화 확인
- **문제점**: 
  - 실시간성 부재 → 늦은 대응 → 손실 확대
  - 데이터 분산 → 인사이트 부재 → 전략적 의사결정 불가
  - 인력 의존 → 확장성 한계 → 규모의 경제 실현 불가
- **기회비용**: 관리에 소요되는 시간을 수익 창출 활동에 투입 불가

[문제 3단계: 미래 경쟁력 상실 위험]
- **로봇택시 시대의 도래** (2030년 예상)
  - 테슬라, 웨이모, 현대차 등 대기업의 자율주행 플랫폼 독점 우려
  - 필요 역량: 실시간 차량 관리, 데이터 분석, AI 기반 운영 최적화
  
- **현재 중소 운수사업자의 준비 부족**
  - 디지털 인프라 부재
  - 데이터 수집/분석 역량 없음
  - 자율주행 시대 운영 경험 전무

- **예상 결과**
  - 중소 사업자 시장 퇴출
  - 대기업 독점 심화
  - 운수업 종사자 ○○만 명 일자리 위협

[문제의 규모와 긴급성]
- 시장 규모: 국내 렌터카 사업자 ○○,○○○개사
- 영향 받는 인구: 종사자 ○○만 명, 이용자 ○○만 명
- 시급성: 로봇택시 상용화까지 약 5년, 지금 준비하지 않으면 늦음
```

**작성 팁:**
- 정량적 데이터로 문제의 심각성 입증 (통계청, 국토교통부 자료 활용)
- 현재 문제 → 미래 위기로 자연스럽게 연결
- "이 문제를 해결하지 않으면..." 식의 위기감 조성

**필요한 데이터 조사:**
- 렌터카 사업자 수, 평균 보유 차량 대수
- 도난/사고/미납 관련 통계
- 로봇택시 시장 전망 (시장조사기관 자료)
- 자율주행 상용화 타임라인

## 3. 우리의 접근 방법 (Solution)

### 작성 프롬프트:

```
Pangea의 솔루션을 다음 3단계로 설명하세요:

[Phase 1: 현재 문제 해결 (2026-2027)]

**3.1 RPA 기반 자동화 시스템**
- **기술 스택**:
  - RPA (Robotic Process Automation): 반복적 업무 자동화
  - IoT 차량 단말기: 실시간 위치/상태 모니터링
  - AI 기반 리스크 분석: 사고/도난 예측 모델

- **핵심 기능**:
  1. 차량 도난 방지
     - 실시간 위치 추적
     - 비정상 이동 패턴 감지 → 즉시 알림
     - 경찰 신고 자동화
     - 효과: 도난 대응 시간 ○○% 단축

  2. 사업 데이터 자동 수집/분석
     - 운행 데이터, 매출, 정비 이력 자동 집계
     - 사업 건전성 점수 산출
     - 금융사 제출용 리포트 자동 생성
     - 효과: 대출 승인율 ○○% 증가

  3. 리스크 사전 예측
     - 사고 위험 운전자 식별
     - 정비 필요 시점 예측
     - 과태료 자동 확인 및 알림
     - 효과: 사고율 ○○% 감소, 운영비 ○○% 절감

  4. 미납금 관리 자동화
     - 연체 자동 감지 및 알림
     - 단계별 독촉 자동화
     - 법적 대응 가이드 제공
     - 효과: 회수율 ○○% 증가

**3.2 차별점 (경쟁사 대비)**
- 기존 차량관리 시스템: 단순 GPS 추적
- Pangea: 데이터 수집 → 분석 → 예측 → 자동 대응 전체 프로세스

**3.3 도입 효과 (ROI)**
- 관리 시간: ○○% 절감
- 운영 비용: ○○% 절감
- 매출: ○○% 증가 (차량 가동률 향상)
- 금융 접근성: 대출 한도 ○○% 증가

[Phase 2: 플랫폼 생태계 구축 (2028-2029)]

**3.4 양면 플랫폼 전략**
- **공급측 (운수사업자)**:
  - 차량 관리 SaaS 제공
  - 데이터 기반 컨설팅
  - 공동구매를 통한 비용 절감

- **수요측 (금융사, 보험사, 정비업체)**:
  - 검증된 사업자 데이터 제공
  - 맞춤형 상품 연결
  - 거래 수수료 수익 모델

**3.5 네트워크 효과**
- 사업자 증가 → 데이터 품질 향상 → 금융사 참여 확대 → 사업자 혜택 증가
- 선순환 구조로 시장 지배력 강화

[Phase 3: 로봇택시 플랫폼 전환 (2030+)]

**3.6 자율주행 시대 대비 역량**

현재 구축 중인 역량:
1. **실시간 차량 관리 인프라**
   - 현재: 유인 차량 모니터링
   - 미래: 자율주행 차량 Fleet 관리

2. **데이터 분석 및 AI**
   - 현재: 리스크 예측, 운영 최적화
   - 미래: 자율주행 경로 최적화, 수요 예측

3. **운영 자동화 경험**
   - 현재: RPA 기반 업무 자동화
   - 미래: 완전 무인 운영 시스템

4. **금융 연계 모델**
   - 현재: 차량 구매 대출 연결
   - 미래: 로봇택시 차량 투자 플랫폼

**3.7 로봇택시 플랫폼 전환 전략**

Target: 중소 운수사업자가 로봇택시를 소유/운영할 수 있는 인프라 제공

제공 가치:
- **차량 관리**: 자율주행 차량 원격 모니터링 및 제어
- **운영 최적화**: AI 기반 배차, 충전, 정비 스케줄링
- **금융 지원**: 차량 구매/리스 연결, 수익 기반 대출
- **규제 대응**: 자율주행 관련 법적 요구사항 자동 관리

차별점:
- 테슬라/웨이모: 자체 차량 + 자체 운영 (폐쇄형)
- Pangea: 다양한 제조사 차량 + 독립 사업자 운영 (개방형)
- → 안드로이드 vs 애플 전략

**3.8 실현 가능성 근거**
- 현재 렌터카 관리 = 미래 로봇택시 관리의 기본 프레임워크
- 5년간 쌓은 데이터와 운영 노하우가 핵심 자산
- 파트너십: 자율주행 제조사, 통신사, 금융사 협력 체계 구축
```

**작성 팁:**
- 단계별로 명확하게 구분하되, 자연스러운 진화 과정 강조
- 기술 용어는 간단한 설명 추가
- 각 단계의 성공 지표(KPI) 명시
- 로봇택시 전환이 "허황된 꿈"이 아닌 "자연스러운 진화"임을 강조

## 4. 트랙션 (Traction)

### 작성 프롬프트:

```
현재까지의 성과와 향후 계획을 다음 구조로 작성하세요:

[이미 달성한 것들]

**4.1 제품 개발 현황**
- ✅ MVP 개발 완료 (pangea.autos)
- ✅ 핵심 기능 구현:
  - 실시간 차량 위치 추적
  - 자동 리스크 알림
  - 사업 데이터 대시보드
  - 금융사 리포트 자동 생성
- ✅ 베타 테스트 진행 중

**4.2 시장 검증**
- 파일럿 고객: ○○개사
- 테스트 차량: ○○대
- 사용자 피드백: "○○○○" (긍정적 평가 인용)
- 문제 해결 지표:
  - 관리 시간 ○○% 절감
  - 리스크 대응 속도 ○○% 향상

**4.3 파트너십**
- 차량 단말기 공급사: Geotab 등과 협력
- 금융사: ○○은행, ○○저축은행과 MOU 체결 (또는 협의 중)
- 정비 네트워크: ○○개 정비소와 제휴

**4.4 대외 인지도**
- 랜딩 페이지 방문자: 월 ○○명
- 데모 신청: ○○건
- 미디어 노출: ○○ (있다면)

[6개월 내 목표]
- 정식 출시 및 유료 고객 ○○개사 확보
- MRR (월 반복 매출) ○○만원 달성
- 금융사 제휴 ○개사 완료
- 추가 파트너십 ○건 체결

[12개월 내 목표]
- 고객사 ○○○개사 확보
- ARR (연간 반복 매출) ○억원 달성
- Break-even 달성
- Series A 투자 유치

[24개월 내 목표]
- 렌터카 시장 점유율 ○○% 확보
- 택시/화물 등 타 운수업 진출 준비
- 해외 진출 (동남아시아 1개국)

**4.5 견인력 증명 지표**
- 고객 획득 비용 (CAC): ○○원
- 고객 생애 가치 (LTV): ○○원
- LTV/CAC 비율: ○○배 (3배 이상이 이상적)
- 월간 성장률 (MoM): ○○%
- 고객 이탈률 (Churn): ○○%
```

**작성 팁:**
- 있는 그대로 정직하게 작성 (과장 금지)
- 작은 성과라도 구체적 숫자로 제시
- 아직 없는 지표는 "예상" 또는 "목표"로 명시
- 트렌드(성장세)를 보여주는 것이 중요

**현재 기준으로 작성 가능한 트랙션:**
- MVP 개발 완료
- 랜딩 페이지 운영 중
- Google Analytics/Clarity 퍼널 분석 시스템 구축
- (추가로 확보한 것들 기재)

## 5. 시장규모 (Market Size)
